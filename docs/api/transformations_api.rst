``candl.transformations``
====================================

.. toctree::
   :hidden:
   :titlesonly:
   :maxdepth: 1

   transformations_common_api
   transformations_common_lensing_api

The transformations module contains:

* ``candl.transformations.common``: common transformations for :math:`TT/TE/EE` spectra.
* ``candl.transformations.common_lensing``: common transformations for :math:`\phi\phi/\kappa\kappa` spectra.
* ``candl.transformations.abstract``: abstract base classes for transformations.
